The Story

CHAPTER 01 — ORIGIN

I don’t just study tech.
I build with it.

Started at LPU with a soldering iron and a secondhand Arduino. While everyone else was memorizing theory, I was flashing firmware onto microcontrollers at 2 AM — debugging signal noise by ear. That restlessness became a compass: if it can be built, it should be built tonight.

CHAPTER 02 — THE BRAND

HustleSmith Studio:
Denim is our canvas.

Luxury custom denim — every jacket hand-painted and embroidered in collaboration with LPU’s Fine Arts talent. No factory lines, no mass production. Each piece is a one-of-one artifact that merges streetwear edge with handcraft tradition.

CHAPTER 03 — THE PLATFORM

ApplifyNow: URL in,
native app out.

Serving as CEO & CTO — built the automated pipeline that converts any website URL into production-ready iOS & Android apps. No code rewrite, no SDK lock-in. The entire conversion engine runs serverless, and I wrote every line of the backend and the React Native bridge.

CHAPTER 04 — THE STACK

Full stack by day,
red team by night.

React, Node, Python, Postgres, Docker — the usual suspects. But the part that keeps me up at night is the cybersecurity rabbit hole: penetration testing, network forensics, and CTF competitions. I break things methodically so I can build them stronger.
